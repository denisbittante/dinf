\section*{Abstract}


%Background, 
Die Enterprise Software ist im Wandel und Microservices werden immer häufiger eingesetzt. 
% Objectives 
\noindent
Diese Arbeit widmet sich der Erforschung der geeigneten Open Source Reporting Engine (OSRE) im Hinblick auf mögliche Ursachen für Peformance-Engpässe und deren Einsatz im Umfeld von Container / Infrastructure as a Service (IaaS). 

%Methods, 
\noindent
Die OSRE wurden in Prototypen genutzt, um verschiedene PDFs zu  erstellen. Die Prototypen wurden als REST-Services auf IaaS deployed und gegen einen Lasttest laufen gelassen. Die Tests gaben Aufschluss über Latenzzeit, Memory-Verbrauch und  Verfügbarkeit des Service. 

%Results, and 
\noindent
RESULTATE FEHELEN NOCH !!! 

%Conclusions
\noindent
Die Experimente deuten darauf hin, dass die Auswahl eines geeigneten Frameworsk eine zentrale Rolle spielt, wenn eine hohe Performance erreicht werden soll.
